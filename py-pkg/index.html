奔向远方

前言

py文件打包成exe，就是说将Python程序打包成应用程序，不在只是几十行代码了，是一个可以运行的小应用了。

就拿我之前写的python爬虫文件，打包后就可以当做应用程序运行了。

pyinstaller

使用 PyInstaller 将 Python 程序生成可直接运行的程序，这个程序就可以被分发到对应的 Windows 或 Mac OS X 平台上运行。
